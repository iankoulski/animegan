# animegan
Create an Anime from your photo using Docker, AnimeGAN, and the do-framework

## Configure

```
./config.sh
```

## Build

```
./build.sh
```

## Run

```
./run.sh
```

## Logs

```
./logs.sh
```

## Stop

```
./stop.sh
```
